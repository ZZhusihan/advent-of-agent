本仓库是我们技术小组记录 Advent of Agents 挑战的学习、挑战和解决方案的共享空间。欢迎每个人贡献自己的笔记和见解！

## 如何贡献

### 创建新一天的笔记

**选项 1：使用脚本（推荐）**
```bash
# 使用 make
make day3

# 或直接使用 Python
python3 make-day.py 3
```

**选项 2：手动创建**
1. 复制 `notes/template.md` 创建新一天的笔记（例如，`notes/day-03.md`）
2. 填写你的学习、挑战、代码示例和资源
3. 更新此 README 以添加指向新笔记文件的链接
4. 提交并推送你的更改！

## 笔记结构

每天的笔记遵循一致的结构：
- **挑战链接**: 当天挑战的直接链接
- **关键学习**: 你学到的东西
- **面临的挑战**: 遇到的问题
- **代码示例**: 相关代码片段
- **资源**: 有用的链接和文档
- **问题/开放话题**: 未解决的问题或讨论点
